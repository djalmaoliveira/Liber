Function html_style_

Set or return CSS style content to use on head tags. This helper could be used inside template files.

Usage:
    // set style content
    html_style_( '<style>h1 {color:green;}</style>' );
    // set style content replacing any existent content
    html_style_( '.alert {color:red;}', true );
    // print stored content
    html_style_();
