method tableData [line 131]

	

    

    

		Array tableData(
String
$tables, [String
$destFolder = ''])
    

    


	
		Return an array of SQL Insert code of each data retrieved from $tables specified, by table name.

If specified $destFolder, each table data will be written on a file like 'tableName.sql' on this folder and return the path to this file instead of data.    Remember that you have to remove the files written after its use.    Usage:  ->tableData('customer'); // return Array('customer'=>'INSERT ...');            ->tableData('customer', '/home/user/sql/'); // written on '/home/user/sql/customer.sql'            ->tableData('customer', 'temp/'); // written on 'APP_PATH/temp/customer.sql'
