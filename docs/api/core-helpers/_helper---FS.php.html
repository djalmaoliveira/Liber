string fs_relative_path_( string $source_path, string $dest_path)

Return a relative path from $dest_path to $source_path specified.

The path returned is relative a first parent directory from $dest_path. Example: $dest_path = '/home/user/myfolder/dest_path'; $source_path = '/home/user/oldfolder/source'; The path will consider that you are on '/home/user/myfolder' and will return '../oldfolder/source'.
