String dt_diffsince_( mixed $past, mixed $future, [integer $detail_level = 0])

Return a textual date/time between two dates $past and $future specified.

$past and $future can be a text like '2010-01-10 15:20:25' or as a seconds like 1263144025. This function consider one month with 30 days. $detail_level show more detailed text from range 1 to 6.
