Input Class

This class is used to get requested data, like post, get, cookies and file.

This class is loaded by default.

Code example:


    // show POST data
    echo Input::post('name');
    echo Input::post('email');

    // show GET data
    echo Input::get('name');
    echo Input::get('email');

    // show COOKIE data
    echo Input::cookie('name');
    echo Input::cookie('email');

    // show FILE data
    echo Input::file('image1');
    echo Input::file('image2');

    // to get a Array of values call without a parameter
    print_r( Input::get() );
