Log Class

This class is used to log messages gerated during normal or not expected working of application.

By default all messages will be write to file on default LOG_PATH in production mode (PROD).

If application mode is DEV, the framework will open a popup window with result of error or exception catched.

Code example:

    // Login simple menssage
    Liber::log()->add('This is a message log.');
    
    // Specifing a level of message
    Liber::log()->add('This is a message log.', 'error');
    Liber::log()->add('This is a message log.', 'info');
    Liber::log()->add('This is a message log.', 'whatever');
    
    // Actually, you can get the object instance to work where you want
    $oLog = Liber::log();
    $oLog->add('Another message');
