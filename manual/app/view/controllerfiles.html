Controller Files

The controller files are stored on controller/ folder.

This files are PHP classes extending Controller class, that manipules http requests and do something, e.g. load view files.

For every controller file is recommended create a index() method, that is a default behaviour from routing mechanism.

Tips: if you want load a default classes or resources for each Controller load, you can use the __construct() method to do this.
