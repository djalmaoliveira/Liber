Cart Class

With Cart class you can manage a generic Cart System.

A Cart is a way where you can put 'things' inside a cart like a product with its id, quantity, price and so on.

Code example:

    // Cart load
    Liber::loadClass('Cart');

    // You can still specifies a 'namespace' as a option to differents carts.
    $oCart = new Cart('namespace');

    // Storing values of 'things'
    $item1 = $oCart->insert( Array('#'=>369, 'product'=>'LiberManual', 'price'=>10.10) );
    $item2 = $oCart->insert( Array('#'=>2549, 'product'=>'Support Service - month', 'price'=>15.99) );

    // replacing existing item
    $oCart->insert( Array('#'=>2549, 'product'=>'Support Service - month', 'price'=>13.99), $item2 );

    // Get entire item
    print_r( $oCart->get($item1) ) ;

    // Get only field of item
    echo $oCart->get($item2, 'product');

    // Search a item by key/value and return item id
    echo $oCart->search( Array('#'=>'2549') );

    // Clear specified item in cart
    $oCart->clear($item1);

    // Clear all cart
    $oCart->clear();
