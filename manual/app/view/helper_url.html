Url Helper

This helper will help you to create URLs.

These helpers are parsed by URL filter.

Code example:

    Liber::loadHelper('Url');

    // relative Url
    url_to_('/manual/helper_form');
    


    // external Url
    url_to_('https://github.com/djalmaoliveira/Liber');
    


    // current Url
    url_current_();
    


    // assets Url
    url_asset_('/js/main.js');
    


    // module assets Url
    url_module_name_asset_('blog'); // set module name
    url_module_asset_('/css/main.css');
    


    // clean a URL parsing to valid URL
    url_clean_('http://blog/post/my first post.html');
