Http Class

This class is used to get requested data, like post, get, cookies and file.

This class is loaded by default.

Code example:


    // show POST data
    echo Http::post('name');
    echo Http::post('email');

    // show GET data
    echo Http::get('name');
    echo Http::get('email');

    // show COOKIE data
    echo Http::cookie('name');
    echo Http::cookie('email');

    // show FILE data
    echo Http::file('image1');
    echo Http::file('image2');

    // Download a file from direct path
    Http::download('/path/to/file');

    // Download a file from direct path, with options
    Http::download('/path/to/file.pdf', array('type' => 'pdf', 'name' => 'report.pdf'));

    // Download a file specifing the 'content' as String and options.
    Http::download(array('content' => 'this is the content file.', 'type' => 'plain', 'name' => 'report.txt'));
