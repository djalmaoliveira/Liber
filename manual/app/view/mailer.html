Mailer Class

This is a simple mailer class with essential methods to send mails.

Code example:

    $oMailer = Liber::loadClass('Mailer',true);

    $oMailer->addTo(    Http::post('to') );
    $oMailer->from(     Http::post('from') );
    $oMailer->reply(    Http::post('reply') );
    $oMailer->subject(  Http::post('subject') );
    $oMailer->body(     Http::post('body') );
    $oMailer->file(     '/path/to/file' ); // attachments
    $oMailer->html(     true );

    $oMailer->send();
