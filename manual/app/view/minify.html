Minify Class

With Minify class you can reduce the size of HTML, CSS or JavaScript files, it can be used, for example, to caching purpose.

This class use a code which was developed from 3rd party programmers.

  • Minify_HTML: Ryan Grove < ryan@wonko.com > and Steve Clay < steve@mrclay.org >
  • CSSmin: Tubal Martin - http://tubalmartin.me/
  • JSMinPlus: Tino Zijdel < crisp@tweakers.net >

Code example:

    // Class load
    Liber::loadClass('Minify');

    // load HTML file
    $html_file = file_get_contents('file.html');
    // output Minified version
    echo Minify::html($html_file);

    // load CSS file
    $css_file = file_get_contents('file.css');
    // output Minified version
    echo Minify::css($css_file);

    // load JavaScript file
    $js_file = file_get_contents('file.js');
    // output Minified version
    echo Minify::js($js_file);
